数据库

数据库分类：

一丶关系型数据库 （表格 信息数量被定性 运行一个服务 存在多个文件):mysql （ mariadb ） sql

sql server/mssql access postgresql db2 一个文件，小数据,无表：access sqllite

二丶非关系型数据库 ( 随意写入数据 无硬性要求): mongodb redis （一般做同步服务器，数据一般不重要）
php采用的是mysql or mariadb

数据库的部分基本命令：

show databases;查看有那几个数据库

create database wordpress;创建一个数据库名字为wordpress

drop database wordpress;删除名字为wordpress的数据库

use test ;使用test数据库

show tables;查看表

drop tables test ; 删除test表
